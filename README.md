# ZL_Counter_jQuery

#使用說明
jQuery CDN

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

counter CDN

    <script src="https://afei80304.github.io/ZL_Counter_jQuery/counter.js"></script>

添加屬性

    data-ct-number="數字"
    data-ct-duration="時間"
